# js-weather-app
A simple weather app that fetches real-time weather data using a public API. This is built to practice API integration and basic Frontend Development. 

# Features
- Search weather by city name
-  Display temperature, and humidity
-  Featches real-time data from public weather API
-  Simple and clean user interface

# Technologies Used
- HTML
- CSS
- JavaScript
- Weather API (OpenWeatherMap)

# Purpose
- This project is for learning purposes only and is part of my journey in learning frontend and handling API-handling skills.
